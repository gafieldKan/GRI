# GRI - 点餐系统

> 基于 awesome-lowcase 低代码平台构建的餐饮行业应用系统

[![License](https://img.shields.io/badge/license-ISC-blue.svg)](LICENSE)
[![React](https://img.shields.io/badge/React-19-blue.svg)](https://reactjs.org/)

## 项目简介

GRI 是一个基于 awesome-lowcase 低代码平台构建的餐饮行业应用系统，包含完整的点餐、支付、开票等功能。

## 快速开始

```bash
git clone https://github.com/gafieldKan/GRI.git
cd GRI
npm install
npm run dev
```

访问 http://localhost:5173/restaurants

## 功能模块

- 商家列表
- 菜单浏览
- 购物车
- 订单管理
- 支付功能
- 发票功能

## 技术栈

- React 19
- Ant Design 6
- Vite 8
